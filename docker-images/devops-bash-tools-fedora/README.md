See a list of included programs with descriptions on the [GitHub project page](https://github.com/KhulnaSoft/DevOps-Bash-tools).

Running the docker container without arguments will show just the list of programs:

```
docker run khulnasoft/bash-tools:fedora
```

Run any given program:

```
docker run khulnasoft/bash-tools:fedora <program> <args>
```
